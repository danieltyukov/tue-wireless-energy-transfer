Inputs
  DCDC input voltage -- 1 analog pin 
  DCDC input current -- current sensor in 1 analog pin
  DCDC output voltage -- 1 analog pin
Outputs
  unipolar PWM to DCDC mosfet driver 
  complementary unipolar PWM with deadtime to DCAC mosfet drivers

PINS on nucleo
A5 - PWM dcdc
A0 - analog pin
A1 - analog pin
A2 - analog pin
A6 - PWM dcac
D9 - PWM* dcac
